### 🔑 ENDPOINTS PRINCIPALES:

**PRODUCTOS (8 operaciones)**
```
POST   /api/admin/products              → Crear
GET    /api/admin/products              → Listar
GET    /api/admin/products/{id}         → Obtener uno
GET    /api/admin/products/active       → Solo activos
PUT    /api/admin/products/{id}         → Editar
PATCH  /api/admin/products/{id}/activate    → Activar
PATCH  /api/admin/products/{id}/deactivate  → Desactivar
DELETE /api/admin/products/{id}         → Eliminar
```

**CATEGORÍAS (5 operaciones)**
```
POST   /api/admin/product-categories      → Crear
GET    /api/admin/product-categories      → Listar
GET    /api/admin/product-categories/{id} → Obtener uno
GET    /api/admin/product-categories/active → Solo activas
PUT    /api/admin/product-categories/{id} → Editar
DELETE /api/admin/product-categories/{id} → Eliminar
```

### ✅ EJEMPLO RÁPIDO (JavaScript):

```javascript
// 1. Obtener token (al login)
const token = 'eyJ...';

// 2. Obtener todos los productos
const productos = await fetch('http://localhost:8081/api/admin/products', {
  headers: { 'Authorization': `Bearer ${token}` }
}).then(r => r.json());

console.log(productos);

// 3. Crear producto
const nuevoProducto = await fetch('http://localhost:8081/api/admin/products', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${token}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    name: 'Hamburguesa',
    description: 'Deliciosa hamburguesa',
    price: 25000,
    categoryId: 1,
    recipeId: 5,
    active: true
  })
}).then(r => r.json());

console.log('Creado:', nuevoProducto);
```

### ⚠️ CÓDIGOS DE ERROR:

| Código | Significado |
|--------|------------|
| 201 | Recurso creado exitosamente |
| 200 | OK, operación exitosa |
| 400 | Datos inválidos (ej: precio negativo) |
| 401 | Token inválido o expirado |
| 403 | Sin permisos (no eres ADMIN) |
| 404 | Recurso no encontrado |
| 500 | Error del servidor |

- **¿Necesito un categoryId?** Sí, debe existir antes de crear producto

- **¿Cómo ordeno resultados?** Usa `?sort=name,asc` o `?sort=price,desc`
- **¿Cómo pagino?** Usa `?page=0&size=10`
